# Week 3: Network Traffic Analysis with Wireshark and Splunk

## Objective
To analyze network traffic and detect suspicious activity using Wireshark and Splunk.

## Tools Used
- Kali Linux VM
- Windows 10 VM
- Wireshark
- Splunk

## Project Plan
1. Install Kali Linux VM
2. Capture traffic using Wireshark
3. Generate network activity
4. Export PCAP files
5. Analyze traffic in Splunk
6. Document findings

## Expected Outcome
Understand how SOC analysts investigate network-based attacks.
